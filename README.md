# Project Stellaris

Sistema de Reservas para Turismo Espacial — FIAP Global Solution 2026/1

##  Descrição

O Project Stellaris é uma plataforma digital de reservas para o mercado de turismo espacial. A aplicação conecta empresas de turismo espacial diretamente ao consumidor, oferecendo pacotes para destinos como Órbita Terrestre, Lua, estação espacial e Marte. Interface minimalista com o foco na experiência do usuário.

## Tecnologias Utilizadas

- React
- Vite
- TypeScript
- Tailwind
- React Router DOM

## Estrutura de Pastas

src/
components/ -  Navbar, Footer, PackageCard
pages/ -  Home, Package, Booking, Reservas, Sobre, FAQ, Integrantes
services/ -  packageService, bookingService
types/ - TypeScript
data/ - dados de pacotes
public/
images/ - imagens dos pacotes
