Project Stellaris:

Sistema de Reservas para Turismo Espacial.

O Project Stellaris é uma plataforma digital (e-commerce/booking de viagens espaciais) desenvolvida para gerenciar a venda de pacotes para o mercado de turismo espacial. A aplicação atua como uma ponte direta entre empresas de turismo espacial e o consumidor.

Aplicação web com estética visual limpa e minimalista, focada na imersão do usuário.
